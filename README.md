# AraucariaRF-OpenMP
## Parallel Random Forest Implementation using OpenMP

This project implements a Random Forest algorithm with OpenMP parallelization for the UFRGS Parallel and Distributed Programming course.

## Project Structure

```
AraucariaRF-OpenMP/
├── src/                    # Source code
│   ├── sequential/         # Sequential Random Forest implementation
│   ├── parallel/          # OpenMP parallel implementation
│   └── utils/             # Utility functions and data structures
├── include/               # Header files
├── tests/                 # Test suites
│   ├── unit/             # Unit tests
│   └── performance/      # Performance test scripts
├── data/                  # Datasets
│   ├── raw/              # Original datasets
│   └── processed/        # Preprocessed datasets
├── scripts/               # Build and execution scripts
│   ├── build/            # Compilation scripts
│   ├── benchmark/        # Performance testing scripts
│   └── vtune/            # Intel VTune profiling scripts
├── results/               # Experimental results
│   ├── performance/      # Performance data and graphs
│   └── profiling/        # VTune profiling results
├── docs/                  # Documentation
│   ├── report/           # Technical report
│   └── presentation/     # Presentation slides
└── dataset/              # Additional datasets (existing)
```
